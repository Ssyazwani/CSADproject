# CSADproject
Wed-Based Application for Booking of Lakepark's sport facilities

### Description
Allows residents to book the facilities and management to manage the facilities of Lakepark

### Software Used

1.Frontend - HTML, CSS, Javascript
2.Backend - Express.js, MySQL

### Mapping

1.Users Access:
 Go to see the facilities available by Lakepark
 Able to see if the facility selected is booked ( not available to book)/ available to book by date/time - (Format should be calendar view )
 Login to view bookings
  1.Name 
  2.Email
  3.Contact Number
  4.Unit Number
 Bookings are made by residents so it can be saved in system:
  1.Name 
  2.Email
  3.Contact Number
  4.Unit Number
  5.When they want to use the facilities (Time)
  6.When they want to use the facilities (Date)
  7.What facilities they want to use
 
2.Management Access:
 Login to view bookings
  1.UserID
  2.Password (Protected/ 12 chars with special characters)
 Able to view the current booking status
  Able to see who book (when,which date and what facilities)
  Able to book and cancel booking

3.About Management Team Page
